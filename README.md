# UTHR — University of Toronto Humanoid Robotics

Official website for the University of Toronto Humanoid Robotics team.

**Live site:** https://t-ilhan.github.io/uthr-website

## About

UTHR is a student-led engineering team at the University of Toronto building Canada's first full-scale bipedal humanoid robot. Founded in 2026, the team operates across six subteams: Mechanical, Electrical, Software, Controls, Business & Sponsorship, and Research.

## Stack

- Pure HTML/CSS/JS — no frameworks
- Hosted on GitHub Pages
- Form submissions via Tally.so

## Updating the Site

1. Edit `index.html` in VS Code
2. Run:
```
git add .
git commit -m "your update message"
git push
```
3. Site updates automatically within ~60 seconds.
